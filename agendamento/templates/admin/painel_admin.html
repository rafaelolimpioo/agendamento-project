Como usar cada página do painel do administrador

Siga estas instruções para entender a função de cada seção e usar o painel com mais segurança, organização e controle.

Home

Esta é a página inicial do administrador. Ela funciona como uma central de orientação e referência rápida para o uso do sistema.

Página inicial
1
Ao entrar como administrador, você será redirecionado primeiro para o Home. Isso ajuda a visualizar o painel e lembrar o que cada área faz.
2
Use esta página para entender a estrutura do sistema antes de navegar para as páginas operacionais da barra lateral.
3
Sempre que houver dúvida sobre o fluxo do painel, volte ao Home para consultar estas instruções.

Serviços

Esta página serve para cadastrar, editar e organizar os serviços oferecidos pelo sistema de agendamento.

Gestão principal
1
Entre em Serviços para visualizar todos os serviços já cadastrados no sistema.
2
Cadastre novos serviços sempre que quiser ampliar o catálogo disponível para os clientes.
3
Edite um serviço quando precisar corrigir nome, descrição, valor ou qualquer outra informação relevante.
4
Exclua um serviço apenas quando tiver certeza de que ele não deve mais aparecer para os agendamentos futuros.

Hoje

Esta área mostra os agendamentos do dia atual e ajuda no acompanhamento da rotina operacional.

Acompanhamento diário
1
Acesse Agendamentos de hoje para ver todos os atendimentos marcados para o dia atual.
2
Use essa página para organizar a rotina do dia, conferir horários ocupados e acompanhar o volume de atendimento.
3
Quando precisar confirmar quem será atendido primeiro ou verificar conflitos de agenda, essa é a primeira página que deve ser consultada.

Agenda

Aqui você acompanha os agendamentos futuros e consegue se antecipar à demanda dos próximos dias.

Planejamento
1
Entre em Próximos para analisar os agendamentos que ainda vão acontecer.
2
Use essa página para se preparar com antecedência, identificar dias mais cheios e planejar melhor sua disponibilidade.
3
Essa área também ajuda a perceber se será necessário bloquear horários, liberar novos períodos ou reorganizar serviços.

Gerenciar horários

Esta é a área responsável pelo controle da disponibilidade da agenda. Nela você consegue bloquear e liberar horários.

Controle de hórarios
1
Acesse Horários sempre que precisar impedir que determinado horário seja reservado.
2
Bloqueie horários em casos como folgas, intervalos, manutenção, eventos externos ou qualquer situação em que o atendimento não poderá acontecer.
3
Libere novamente um horário quando ele voltar a ficar disponível para novos agendamentos.
4
Use essa página com atenção, porque ela interfere diretamente no que o cliente pode ou não pode reservar no sistema.

Dashboard financeiro

Esta página reúne informações de relatório para análise do desempenho recente do sistema.

Relatórios
1
Entre em Dashboard financeiro para visualizar dados consolidados do último período.
2
Use esse relatório para entender o comportamento da agenda, acompanhar volume de atendimentos e observar resultados operacionais.
3
Essa área ajuda na tomada de decisão, porque mostra uma visão mais ampla do sistema além do dia atual.
